Welcome to marvin's documentation!
==============================================
This is a Python library that enables easy arbitrary timing using various hardware from Marvin Test
Solutions, Inc.  Arbitrary timing is a typical requirement for atomic physics
experiements as are supported by the seperate package named
`Arbwave <https://github.com/afrl-coldatoms/arbwave>`_.

See :ref:`Quick Start <quickstart>` for a few examples.
